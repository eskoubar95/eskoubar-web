# Product Requirements Document (PRD)

## Overview
Personligt brand website for "eskoubar" (Nicklas Eskou) der kombinerer content marketing med freelance salgskonvertering.

## Core Features

### 1. Blog System
**Prioritet:** Høj (primært formål)
**Beskrivelse:** 
- Blog funktionalitet til at publicere artikler
- Kategorisering og tagging
- SEO-optimering
- Responsive design

**Acceptance:** Brugere kan læse blog posts, navigere mellem posts, og finde relevant content.

### 2. Newsletter Sign-up System
**Prioritet:** Høj (primært formål)
**Beskrivelse:**
- Newsletter sign-up formular
- Integration med newsletter platform (besluttes)
- Email validation
- Success/error feedback

**Acceptance:** Brugere kan tilmelde sig newsletter og modtage bekræftelse. Email adresser gemmes i newsletter platform.

### 3. Contact Form
**Prioritet:** Medium (sekundært formål)
**Beskrivelse:**
- Kontaktformular til freelance henvendelser
- Felter: navn, email, besked/projektbeskrivelse
- Email notification til Nicklas
- Success/error feedback

**Acceptance:** Brugere kan sende henvendelse. Nicklas modtager email med formular data.

### 4. Brand Presentation Pages
**Prioritet:** Medium
**Beskrivelse:**
- "Om mig" side
- "Services" side
- Navigation og footer
- Responsive design

**Acceptance:** Brugere kan læse om Nicklas, se services, og navigere mellem sider.

### 5. Portfolio/Case Studies (Conditional)
**Prioritet:** Lav (conditional - besluttes før launch)
**Beskrivelse:**
- Portfolio/case studies sektion
- Visuelle eksempler på projekter
- Projektbeskrivelser

**Acceptance:** Brugere kan se eksempler på tidligere arbejde (hvis inkluderet på launch).

## Non-Functional Requirements

### Performance
- Siden skal loade hurtigt (< 3 sekunder)
- Responsive på mobile, tablet, desktop

### SEO
- SEO-optimerede meta tags
- Sitemap
- Struktureret data (schema.org)

### Accessibility
- Grundlæggende accessibility (WCAG 2.1 Level A minimum)

### Security
- Formular validation
- Spam protection (CAPTCHA eller lignende)
- Secure form submissions

## Out of Scope (MVP)
- Booking system
- CRM integration
- E-commerce
- Medlemskabsområder
- Multi-user systemer
- Avancerede analytics (beyond basic)
