# Project Milestones

## M1: Foundation & Setup
**Objective:** Etablere teknisk fundament og development environment.

**Scope:**
- In: Teknisk stack valg, hosting/domain setup, project structure, development environment
- Out: Feature implementation, content creation

**Exit Criteria:**
- Teknisk stack valgt og dokumenteret
- Development environment opsat og fungerende
- Hosting og domain konfigureret
- Basic project structure på plads
- Kan deploye "Hello World" til production

**Dependencies:** None

---

## M2: Core Infrastructure
**Objective:** Bygge grundlæggende website infrastructure og design system.

**Scope:**
- In: Basic routing, navigation, responsive layout, styling/branding, footer
- Out: Blog system, forms, content pages

**Exit Criteria:**
- Website kan vises i browser
- Responsive design fungerer (mobile, tablet, desktop)
- Navigation og routing fungerer
- Basic styling og branding på plads
- Footer med links implementeret

**Dependencies:** M1

---

## M3: Content System (Blog)
**Objective:** Implementere blog funktionalitet til content publishing.

**Scope:**
- In: Blog listing page, individual post pages, SEO meta tags, content management
- Out: Newsletter integration, contact form

**Exit Criteria:**
- Blog system fungerer (create, read, list)
- Blog posts kan vises individuelt
- SEO meta tags implementeret
- Content management workflow etableret
- Minimum 1 test blog post publiceret

**Dependencies:** M2

---

## M4: Newsletter Integration
**Objective:** Implementere newsletter sign-up system (primært formål).

**Scope:**
- In: Newsletter platform valg, sign-up formular, email validation, integration
- Out: Contact form, portfolio

**Exit Criteria:**
- Newsletter platform valgt og konfigureret
- Sign-up formular fungerer
- Email validation og error handling
- Success feedback til bruger
- Integration testet end-to-end (test signup virker)

**Dependencies:** M2

---

## M5: Contact & Brand Pages
**Objective:** Implementere kontaktformular og brand præsentation (sekundært formål).

**Scope:**
- In: Contact form, "Om mig" page, "Services" page, spam protection
- Out: Portfolio/case studies (conditional)

**Exit Criteria:**
- Kontaktformular fungerer
- Email notification til Nicklas fungerer
- "Om mig" og "Services" sider implementeret
- Formular spam protection på plads
- Test submission virker

**Dependencies:** M2

---

## M6: Polish & Launch
**Objective:** Færdiggøre, optimere og launch website.

**Scope:**
- In: SEO optimering, performance tuning, content creation (3-5 blog posts), final testing, launch
- Out: Portfolio (hvis ikke inkluderet), future enhancements

**Exit Criteria:**
- Alle features testet og fungerende
- SEO optimering gennemført (meta tags, sitemap, structured data)
- Performance optimering gennemført (< 3s load time)
- Minimum 3-5 blog posts publiceret
- Responsive design verificeret på alle devices
- Website er live og tilgængelig
- Ready for production launch

**Dependencies:** M3, M4, M5

---

## Notes
- M3, M4, og M5 kan delvist arbejdes på parallelt efter M2 er færdig
- Portfolio/case studies er conditional - besluttes før M6
- Newsletter platform valg skal ske tidligt i M4
